## 📱 CONNECT YOUR EXPO APP

### Step 1: Find Your Computer IP
```bash
ipconfig getifaddr en0    # macOS/Linux
ipconfig                  # Windows (look for IPv4)
```
Example output: `192.168.1.100`

### Step 2: Create `.env` in Your Expo App
```bash
EXPO_PUBLIC_BACKEND_URL=http://192.168.1.100:3000
```

### Step 3: Use in Your Components
```typescript
import axios from 'axios';

const api = axios.create({
  baseURL: process.env.EXPO_PUBLIC_BACKEND_URL
});

// Search Quran
const response = await api.get('/api/quran/search', {
  params: { query: 'allah' }
});
```

## 🎯 API ENDPOINTS OVERVIEW

### Quran (5 endpoints)
```
GET  /api/quran/search          Search verses
GET  /api/quran/surah/:id       Get full Surah
GET  /api/quran/verse/:s/:v     Get single verse
GET  /api/quran/surahs          Get all Surahs
GET  /api/quran/editions        Get languages
```

### AI Scholar (3 endpoints)
```
POST /api/ai/ask                Ask a scholar
GET  /api/ai/scholars           List scholars
POST /api/ai/batch              Batch questions
```

### Hadith (4 endpoints)
```
GET  /api/hadith/search         Search Hadith
GET  /api/hadith/:id            Get Hadith
GET  /api/hadith/collections    List collections
GET  /api/hadith/theme/:theme   Search by theme
```

### User (6 endpoints)
```
GET  /api/user/profile          Get profile
GET  /api/user/settings         Get settings
PUT  /api/user/settings         Update settings
GET  /api/user/favorites        Get favorites
POST /api/user/favorites        Add favorite
GET  /api/user/stats            Get statistics
```
